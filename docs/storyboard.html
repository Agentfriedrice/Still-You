STILL YOU

storyboard & system flowchart · vis 160a
An interactive 2D pixel-art web experience where the user walks through a dark, memory-like hallway, encountering objects from the past, and makes small choices that shape a reflective ending about growth, regret, care, and identity.

The Arc

Four hallways, four years. Each one ends in a doorway. The doorway reflects what the player did and lets them step through to the next.

Visual Language

A 4:3 boxed frame used to invoke a sense of nostalgia. Side-scrolling hallway with doors along the back wall, floor tiles, soft overhead lamp pools, drifting dust particles, and a CRT scanline overlay. These elements allow the experience to feel like an old game cartridge that remembers you.
Common elements (used for every year):
  • 4:3 boxed viewport — containment. Memories arrive framed.
  • CRT scanlines + thin vignette frame — nostalgia, an old screen vibe.
  • Doors along the back wall — The hallway is somewhere, not nowhere.
  • Overhead lamp pools — soft elliptical light. Anchors the player to the architecture.
  • Floor tile lines — short stripes at the front edge of the floor; reads as perspective.
  • Drifting dust particles — tinted by the year's palette. Slow, ambient motion.
  • Memory glows — soft colored ellipse under each interactable. Brightens after a choice.
  • "?" → "·" — the hint above each memory swaps to a quiet dot once the player has chosen.
  • Doorway threshold — a warm vertical line in a faint frame at the right edge, with a small "step through" hint when ready.
